# Snake and Ladder
This final project is aimed to implement the Snake and Ladder game.

# Build and Run

` mkdir SnakeAndLadders 
 clone github.com/natitek/final_project 
cd final_project 
g++ main.cpp -o game.exe -I src/include -L src/lib -lraylib 
 ./game.exe`

## Features
  - Single Player mode
  - Multiplayer Mode
  - Settings
  - Profiles
